# NftClubX

NftClubX is a community-driven NFT marketplace built on opBNB, featuring automated buyback & burn, fair FIFO queues, and true ownership.

## Features

- **True Transferability**: Freely transfer or list NFTs on major marketplaces.
- **Auto Buyback & Burn**: A portion of every transaction is burned to reduce supply.
- **Transparent FIFO Queue**: Fair first-in, first-out sale queue.
- **Intelligent Tokenomics**: Self-sustaining deflationary economy.
- **Referral System**: 16-level referral system to unlock deeper levels.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.
